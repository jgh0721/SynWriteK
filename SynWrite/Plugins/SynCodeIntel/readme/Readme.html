SynCodeIntel plugin

SynCodeIntel

Intro

Intelligent auto-completion plugin for SynWrite editor, for several lexers: Python (2 and 3), PHP, Perl, Ruby, JavaScript, Tcl, SCSS, LESS.
Based on CodeIntel2 library from ActiveState (it's also used in Komodo Edit).

How to install

  • create folder "SynCodeIntel" inside "Plugins" subfolder of SynWrite install folder
  • unpack plugin package into this folder ("SynCodeIntel.dll" file must exist directly under "Plugins\SynCodeIntel" folder)
  • in SynWrite open menu command "Options -- Advanced -- Edit file SynPlugins.ini"
  • in opened editor of "SynPlugins.ini" file add a line under section [Complete]: 

    [Complete]
SynCodeIntel=SynCodeIntel\SynCodeIntel.dll;PHP,PHP (dev),Python,Perl,Ruby,JavaScript,SCSS,LESS,XML,XSLT,Tcl;

  • you may adjust lexers list in the above line, e.g. remove lexers which are not needed

How to use

Note a) First plugin call initializes CodeIntel database, this takes about 5-10 sec. You can delete this database: folder "%userprofile%\.codeintel\db".

Note b) If plugin fails, you should see message in SynWrite statusbar, e.g. "[SynCodeIntel] Cannot parse position". This msg in statusbar is visible only for 1-2 sec.

1) Auto-completion

Open for ex Python script, place caret after "objname." (where objname is some object instance), and press Ctrl+Space (default auto-complete shortcut). You should see message in SynWrite statusbar "Trying auto-complete: .........", then you should see, after 1-2 sec delay, auto-completion list for your objname.

2) Function hints

You can place caret after "objname.funcname(" and press Ctrl+Shift+Space (default func-hint shortcut), after 1-2 sec delay you'll see function hint, above caret, for function "objname.funcname".
