FindIdPas plugin

FindIdPas is find-id plugin for SynWrite text editor, i.e. it can find declaration of identifiers for Pascal lexer. Activate Pascal lexer, right-click any id in the source code, and in the context menu call item "Find ID". Plugin will open location of clicked id declaration (if it can find such declaration).

Code of id search engine is rough and unreliable, it often makes mistakes. This plugin is written mainly for demo purposes.

How to install

  • Unpack plugin files into "Plugins\FindIdPas" subfolder of SynWrite install folder
  • Open SynWrite, call menu item "Options -- Advanced -- Edit file SynPlugins.ini"
  • In opened editor of SynPlugins.ini file add line in section [FindID]: 
    [FindID]
pas=FindIdPas\FindIDPas.dll;Pascal,Pascal script;
  • Adjust lexers list in the above line, if needed
  • Restart program
