Plugin "Tidy XML"

This is command plugin for SynWrite editor.
It allows to format XML source code (lexer "XML"), using Python engine.

Plugin adds command in SynWrite menu "Tools - Plugins".

How to install

Just open plugin's zip-file in SynWrite, and confirm installation.
Default hotkey Ctrl+Alt+X can be changed using SynWrite command "Options - Advanced - Edit file SynPlugins.ini".
