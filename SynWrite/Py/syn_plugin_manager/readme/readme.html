Plugin "Plugin Manager"

This is command plugin for SynWrite editor. It gives two commands:

  • "Edit plugin": this shows plugins menu and opens chosen plugin's source file for editing.
  • "Remove plugin": this works for Python plugins only yet. It shows plugins menu, then unregisters chosen plugin from "SynPlugins.ini" and removes plugin folder into "Py\__trash" subfolder. Then you need to restart SynWrite.

Plugin has submenu in SynWrite menu "Tools - Plugins".

How to install

Just open zip-file with plugin in SynWrite, and confirm installation.
You may set hotkeys in "SynPlugins.ini" file, using SynWrite command "Options - Advanced - Edit file SynPlugins.ini".
